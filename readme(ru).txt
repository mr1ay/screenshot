:: Вы можете сделать снимок экрана
:: Пожалуйста, установите горячие клавиши f7 и f8, чтобы использовать программу
::
::
:: если вы хотите использовать f7,
:: -Если мышь находится в двух разных местах, снимок экрана будет сделан между двумя координатами.
:: так что подведите мышь к координате, затем откройте f7. затем f. затем снова переведите указатель мыши на координату, затем откройте f7
::
:: -Если мышь неподвижна, захватывается весь экран. Дважды нажмите f7 для этого
::
::
:: если вы хотите использовать f8, просто нажмите f8 один раз, и программа сделает снимок экрана
